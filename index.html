What Is It ?

It's a tool to help you give an overview of all the colors your project uses. It gives you the sorted, unique colors but differentiates when it needs to, so you can find and fix. ( eg. #fff vs #ffffff )

How To Use It ?

  • Search for colors in your project. How ?
  • Paste the colors in the box below. Each in a new line please.
  • Hit that damn button! Will you?

It's still a Work In Progress with handful of more features like parsing from CSS & Hue Based Sorting to come.
